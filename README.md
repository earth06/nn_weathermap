# nn_weathermap
search nearest weather date by nn 


## ディレクトリ構成


```
├── docs
├── nn_weathermap
│   ├── data
│   │   └── netcdf 再解析データ置き場
│   ├── etc
│   ├── images
│   │    └── map 再解析データを天気図にした画像置き場
│   │     
│   ├── model
│   │   └── index 天気図探索モデル置き場
│   └── src
│       ├── ai 天気図探索モデルのコード
│       └── app webアプリのコード
├── notebook


```
